A QUICK GUIDE TO SET UP YOUR LINUX SYSTEM

In this guide, we discuss the things to be done after installing your new copy of a Linux distribution, especially Ubuntu and Ubuntu bases distros like Linux Mint and Pop OS.

This article will get updated with time.

The screenshots provided are from a Linux Mint System, they will slightly differ in Ubuntu and Pop OS.

  1. Update software Repository and Install latest Packages

Execute the following command in the terminal.

sudo apt update && sudo apt upgrade -y

  1. Install RESTRICTED EXTRAS  

Many Linux distros don’t bundle media codecs which are essential to play video and audio files due to legal restrictions. Install them by executing the following script in the terminal :

apt install ubuntu-restricted-extras   

  1. Boost Application Load Time with PRELOAD

Boost your frequently used applications by already leading them into your RAM even before you use them. Enable this your system has 8GB of RAM or more.

Note: You won’t see any improvement instantly, Preload learns your app usage pattern over time and only boosts the loading time of the most frequently used apps.

sudo apt install preload

  1. Boost your Website load time by updating your DNS

Connect to a fast DNS like Google DNS to reduce the delay between the instant your search for a website and the instant the actual website starts to load.

Note: This doesn’t actually increase your download or upload speed, it just reduces the initial delay taken to load a website in your browser.

Step 1: Go to system settings, then network settings, find your connected wired or wireless network, and click on the Gear  ⚙️ icon.

Step 2: Under the IPV4 tab, disable the automatic DNS, enter the addresses 8.8.8.8, 8.8.4.4 for setting Google DNS and apply the settings.

  

  1. Install Your favorite Applications

By default, applications can be installed from the Software-Center including VLC media player, Skype, Spotify, WhatsApp, Blender, GIMP, and code editors like VS Code, Sublime Text, etc.

  1. It is not recommended to install the Flatpak packages of applications available in the store. They are very large in size and in most cases slower in performance. Install such packages from alternate sites like the ones mentioned below.

  1. Another popular site for installing packages in Linux is the Snap Store (https://snapcraft.io/store)

  1. Some applications like Google Chrome are not available on the software center or the snap store, you need to install them by manually downloading the installation packages (.deb) from their websites.
